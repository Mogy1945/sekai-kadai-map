# 世界問題課題マップ — 日本編

日本が直面する社会課題を、**実データに基づく深刻度スコア**で可視化したインタラクティブマップ。

暗い地球から日本にズームインすると、課題が蜘蛛の巣状につながったネットワークが現れる。
円の大きさ＝深刻度。課題をタップすると出典付きの詳細が見られる。

- **深刻度スコア** = 影響人数(40%) × 経済インパクト(35%) × 緊急度(25%) — 出典データからの機械的算出
- **大課題→サブ課題**の親子構造 + **課題間の因果リンク**(悪化させる/原因となる/相互に影響…)
- 全数値に**出典リンク付き**。政治的に立場が分かれる論点は両論併記
- オフライン動作の単一HTML。スマホ対応(ピンチズーム/ボトムシート)

## 遊び方

`index.html` をブラウザで開くだけ。

- 地球 → 「日本の課題マップを開く」or 日本をタップ
- ノードをタップ → 詳細パネル(スコア内訳・統計・出典・関連課題)
- ドラッグでノード移動 / 背景ドラッグでパン / ホイール・ピンチでズーム
- 「一覧」= スコア順テーブル / 「大きさの根拠」= スコア算出方法

## データについて

課題の選定・数値は制作者の意見ではなく、以下の手順で構築:

1. **3視点タクソノミー**: 政府世論調査・省庁白書 / OECD等の国際比較 / シンクタンク・報道世論調査 から独立に課題を抽出し統合
2. **課題別調査**: 課題ごとに最新の一次資料から統計・出典を収集
3. **敵対的ファクトチェック**: 別エージェントが全数値・出典URLを一次資料と突合し修正
4. **抜け漏れ批評**: 最終リストへの補完調査

スコア算出式はアプリ内「大きさの根拠」に明記。

## 構成

- `index.html` — 本体(単一HTML・ビルド成果物)
- `src/graph.js` — スコア計算+力学シミュレーション(UI非依存・node直require可)
- `src/data.js` — 課題データセット(出典付き)
- `src/mapdata.js` — 世界地図(Natural Earth由来・等距円筒図法)
- `src/app.js` / `style.css` / `template.html` — UI層
- `build.mjs` — `node build.mjs` で index.html 再生成
- `sim/e2e.cjs` — E2E(データ整合性+物理+jsdom UIフロー)
- `tools/` — 地図変換・データ生成・スクリーンショット

## 開発

```bash
node build.mjs        # ビルド
node sim/e2e.cjs      # テスト
node tools/shots.mjs  # スクリーンショット(要playwright)
```
